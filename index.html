    Berdikari ERP

    Project Overview :

    Berdikari ERP is a custom-built ERP system developed for PT. Berdikari Putra Abadi, a poultry processing company. This ERP system is designed to streamline operations across various departments, from sales to tax reporting.

    My Contribution :

    I developed a majority of the key modules in Berdikari ERP, including:

    • Customer and Courier Web Apps for client and courier interactions
    • Quotation, Sales Order, and Purchase Order modules to manage sales and procurement workflows
    • Invoice and Finance modules to support accurate billing and financial tracking
    • Label Printing for product tracking and packaging
    • E-Faktur for automated tax reporting in CSV form and compliance with Indonesian regulations
    Technology Used :

    Ruby on Rails, PostgreSQL, Bootstrap, AJAX, JQuery, QZ-Tray

    Sanwa ERP

    Project Overview :

    Sanwa ERP is a tailored ERP system designed for PT. Sanwa Antar Nusa, a logistics and expedition company. This system supports Sanwa in managing their complex operations by offering modules for package handling, invoicing, and finance management.

    My Contribution :

    I contributed significantly to this project by developing key features that streamline operations. One major feature is a system for managing packages that need to be split into multiple parts, allowing for partial deliveries while still tracking them under the same order. I also created an automated invoicing module, reducing manual work for the finance team, and took part in building out the finance module for comprehensive financial oversight. Additionally, I served as the main customer support contact for this ERP, assisting users and addressing system queries.

    Technology Used :

    Ruby on Rails, PostgreSQL, Bootstrap, AJAX, JQuery
